# Pharma Commercial Insights Semantic Layer Platform

## Overview

An end-to-end Data Engineering project designed to simulate a modern pharmaceutical analytics platform using Snowflake, dbt, Airflow, Python, SQL, and Semantic Layer architecture.

The project demonstrates how raw operational data can be transformed into trusted business metrics using a Medallion Architecture (Bronze → Silver → Gold).

---

## Business Goals

The platform enables analysis of:

* Revenue Performance
* Territory Insights
* Distributor Performance
* Interaction Effectiveness
* Inventory Health
* Product Performance

---

## Architecture

Source Systems
→ Bronze Layer
→ Silver Layer
→ Gold Layer
→ Semantic Layer
→ Dashboard
